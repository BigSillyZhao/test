
[[_queryusingget]]
==== 获取模型对比状态
....
GET /compare
....


===== 说明
应用发起对比以后，可以通过该接口查询对比状态


===== 参数

[options="header", cols=".^2a,.^3a,.^9a,.^4a,.^2a"]
|===
|类型|名称|说明|类型|默认值
|**Header**|**Authorization** +
__必填__|authorization header|string|`"bearer dc671840-bacc-4dc5-a134-97c1918d664b"`
|**Query**|**compareId** +
__必填__|模型对比ID|integer (int64)|
|===


===== 响应

[options="header", cols=".^2a,.^14a,.^4a"]
|===
|HTTP代码|说明|类型
|**200**|OK|<<_75831995758d85ecb92841b9dd2052f6,GeneralResponse«ModelCompareBean»>>
|**401**|Unauthorized|无内容
|**403**|Forbidden|无内容
|**404**|Not Found|无内容
|===


===== 消耗

* `application/json`


===== 生成

* `application/json`
* `\*/*`


===== HTTP请求示例

====== 请求 path
----
/compare?compareId=0
----


====== 请求 header
[source,json]
----
"string"
----
