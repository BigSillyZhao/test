
[[_getaccesstokenusingpost]]
==== 获取AccessToken
....
POST /oauth2/token
....


===== 说明
在调用其他API之前，必须先获取AccessToken。AccessToken的有效期为7天


===== 参数

[options="header", cols=".^2a,.^3a,.^9a,.^4a"]
|===
|类型|名称|说明|类型
|*Header*|*Authorization* +
_可选_|将字符串 appKey:appSecret 拼接后（中间用冒号连接），对其进行BASE64编码， 然后在编码后的字符串前添加字符串Basic和一个空格， 即：“Basic” + “ ” + Base64Encode(appKey + “:” + appSecret)|string
|===


===== Request Body
[source,json]
----
curl -X DELETE
-H "Authorization: bearer <your accessToken>"
-H "Content-Type: application/json"
"https://api.bimface.com/oauth2/token"
----


===== HTTP请求示例

====== 请求 path
----
/oauth2/token
----


====== 请求 header
[source,json]
----
"string"
----
===== 响应

[options="header", cols=".^2a,.^14a,.^4a"]
|===
|HTTP代码|说明|类型
|*200*|OK|<<_e8a444e639629783389d73039b948b5e,GeneralResponse«AccessTokenBean»>>
|*201*|Created|无内容
|*401*|Unauthorized|无内容
|*403*|Forbidden|无内容
|*404*|Not Found|无内容
|===
