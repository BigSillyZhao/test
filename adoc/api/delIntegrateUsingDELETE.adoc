
[[_delintegrateusingdelete]]
==== 删除集成模型
....
DELETE /integrate
....


===== 说明
根据集成模型id删除集成模型


===== 参数

[options="header", cols=".^2a,.^3a,.^9a,.^4a,.^2a"]
|===
|类型|名称|说明|类型|默认值
|*Header*|*Authorization* +
_必填_|authorization header|string|`"bearer dc671840-bacc-4dc5-a134-97c1918d664b"`
|*Query*|*integrateId* +
_必填_|集成id|integer (int64)|
|===


include::detail/request.adoc[]


===== 生成

* `application/json`
* `\*/*`


===== HTTP请求示例

====== 请求 path
----
/integrate?integrateId=0
----


====== 请求 header
[source,json]
----
"string"
----

[options="header", cols=".^2a,.^14a,.^4a"]

|*204*|No Content|无内容
|*401*|Unauthorized|无内容
|*403*|Forbidden|无内容
|===
