
[[_createtranslateofflinedatabagusingpost]]
==== 文件ID创建离线数据包
....
POST /databag
....


===== 说明
通过文件ID创建离线数据包


===== 参数

[options="header", cols=".^2a,.^3a,.^9a,.^4a,.^2a"]
|===
|类型|名称|说明|类型|默认值
|**Header**|**Authorization** +
__必填__|authorization header|string|`"bearer dc671840-bacc-4dc5-a134-97c1918d664b"`
|**Path**|**fileId(与integrateId，compareId必选一)** +
__可选__|通过文件ID查询离线数据包|Long|
|**Path**|**integrateId(与fileId，compareId必选一)** +
__可选__|通过集成ID查询离线数据包|Long|
|**Path**|**compareId(与fileId，integrateId必选一)** +
|**Query**|**callback** +
__可选__|回调url|string|
|===


===== 生成

* `application/json`
* `\*/*`


===== HTTP请求示例

====== 请求 path
----
curl -X POST
-H "Authorization: bearer <your accessToken>"
-H "Content-Type: application/json"
"https://api.bimface.com/databag?fileId=<your fileId>"

curl -X POST
-H "Authorization: bearer <your accessToken>"
-H "Content-Type: application/json"
"https://api.bimface.com/databag?integrateId=<your integrateId>"

curl -X POST
-H "Authorization: bearer <your accessToken>"
-H "Content-Type: application/json"
"https://api.bimface.com/databag?compareId=<your compareId>"
----
====== 请求 header
[source,json]
----
"string"
----

无
