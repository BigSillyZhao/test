
[[_createintegrateofflinedatabagusingput]]
==== 文件集成ID创建离线数据包
通过文件集成ID创建离线数据包


===== 参数

[options="header", cols=".^2a,.^3a,.^9a,.^4a,.^2a"]
|===
|类型|名称|说明|类型|默认值
|*Header*|*Authorization* +
_必填_|authorization header|string|`"bearer dc671840-bacc-4dc5-a134-97c1918d664b"`
|*Path*|*integrateId* +
_必填_|通过集成模型ID创建离线数据包|integer (int64)|
|*Query*|*callback* +
_可选_|回调url|string|
|===


===== 生成

* `application/json`
* `\*/*`


===== HTTP请求示例

====== 请求 path
----
/integrations/0/offlineDatabag
----
===== Request Body
无
----
curl -X POST
-H "Authorization: bearer <your accessToken>"
-H "Content-Type: application/json"
"https://api.bimface.com/files?fileId={your fileId}/offlineDatabag?callback={callback}"
----

====== 请求 header
[source,json]
----
"string"
----
===== 响应

[options="header", cols=".^2a,.^14a,.^4a"]

|*201*|Created|无内容
|*401*|Unauthorized|无内容
|*403*|Forbidden|无内容
