
[[_previewsharelinkusingget]]
==== 模型预览页面
....
GET /preview/{token}
....


===== 说明
根据分享链接返回模型预览页面


===== 参数

[options="header", cols=".^2a,.^3a,.^9a,.^4a,.^2a"]
|===
|类型|名称|说明|类型|默认值
|**Header**|**Authorization** +
__必填__|authorization header|string|`"bearer dc671840-bacc-4dc5-a134-97c1918d664b"`
|**Path**|**token** +
__必填__|token|string|
|===


===== 响应

[options="header", cols=".^2a,.^14a,.^4a"]
|===
|HTTP代码|说明|类型
|**200**|OK|<<_modelandview,ModelAndView>>
|**401**|Unauthorized|无内容
|**403**|Forbidden|无内容
|**404**|Not Found|无内容
|===


===== 消耗

* `application/json`


===== 生成

* `application/json`
* `\*/*`


===== HTTP请求示例

====== 请求 path
----
/preview/string
----


====== 请求 header
[source,json]
----
"string"
----


===== HTTP响应示例

====== 响应 200
[source,json]
----
{
  "empty" : true,
  "model" : "object",
  "modelMap" : {
    "string" : "object"
  },
  "reference" : true,
  "status" : "string",
  "view" : {
    "contentType" : "string"
  },
  "viewName" : "string"
}
----
