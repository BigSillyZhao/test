
[[_deleteusingdelete]]
==== 删除模型对比状态
....
DELETE /compare
....


===== 说明
删除模型对比


===== 参数

[options="header", cols=".^2a,.^3a,.^9a,.^4a,.^2a"]
|===
|类型|名称|说明|类型|默认值
|*Header*|*Authorization* +
_必填_|authorization header|string|`"bearer dc671840-bacc-4dc5-a134-97c1918d664b"`
|*Query*|*compareId* +
_必填_|模型对比ID|integer (int64)|
|===

===== 生成

* `application/json`
* `\*/*`


===== HTTP请求示例

====== 请求 path
----
/compare?compareId=0
----


====== 请求 header
[source,json]
----
"string"
----
