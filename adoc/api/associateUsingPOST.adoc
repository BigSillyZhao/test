
[[_associateusingpost]]
==== 发起业务挂接
....
POST /association/{businessType}
....


===== 说明
发起业务挂接


===== 参数

[options="header", cols=".^2a,.^3a,.^9a,.^4a,.^2a"]
|===
|类型|名称|说明|类型|默认值
|*Header*|*Authorization* +
_必填_|authorization header|string|`"bearer dc671840-bacc-4dc5-a134-97c1918d664b"`
|*Path*|*businessType* +
_可选_|业务挂接类型|string|
|*Body*|*request* +
_必填_|request|<<_businessassociationrequest,BusinessAssociationRequest>>|
|===

===== 生成

* `application/json`
* `\*/*`


===== HTTP请求示例

====== 请求 path
----
/association/string
----


====== 请求 header
[source,json]
----
"string"
----


====== 请求 body
[source,json]
----
{
    "name": "spring test",
    "integrateId": 1248789977538784,
    "ruleFileIds": [
        1248787101475008
    ],
    "priority": 2,
    "callback": "http://www.bimface.com",
    "config": {
        "sourceFiles": [

        ]
    }
}
----
