
[[_getappendfileusingget]]
==== 查询追加文件信息
....
GET /appendFiles/{appendFileId}
....
根据appendFileId查询追加文件信息。

===== 参数

[options="header", cols=".^2a,.^3a,.^9a,.^4a,.^2a"]
|===
|类型|名称|说明|类型|默认值
|*Header*|*Authorization* +
_必填_|authorization header|string|`"bearer dc671840-bacc-4dc5-a134-97c1918d664b"`
|*Path*|*appendFileId* +
_必填_|appendFileId|integer (int64)|
|*Path*|*name* +
_必填_|append file id|integer (int64)|
|===
===== Request Body
无
[source,json]
----
curl -X GET
-H "Authorization: bearer <your accessToken>"
-H "Content-Type: application/json"
"https://file.bimface.com/appendFiles/{appendFileId}"
----

===== 响应

[options="header", cols=".^2a,.^14a,.^4a"]
|===
|HTTP代码|说明|类型
|*200*|OK|<<_70c9021eb6fd88af1fe87c7da4d0e5d0,GeneralResponse«AppendFileBean»>>
|*404*|Not Found|无内容
|===

===== HTTP请求示例

====== 请求 path
----
/appendFiles/0
----


====== 请求 header
[source,json]
----
"string"
----
