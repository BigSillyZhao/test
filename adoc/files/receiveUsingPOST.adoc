
[[_receiveusingpost]]
==== 处理OSS文件上传回调
....
POST /oss/receive
....


===== 说明
处理OSS文件上传的回调


===== 参数

[options="header", cols=".^2a,.^3a,.^9a,.^4a"]
|===
|类型|名称|说明|类型
|*Header*|*Authorization* +
_可选_|Authorization|string
|*Header*|*x-oss-pub-key-url* +
_可选_|x-oss-pub-key-url|string
|*Query*|*appkey* +
_可选_|appkey|string
|*Query*|*callback* +
_可选_|callback|string
|*Query*|*etag* +
_可选_|文件etag|string
|*Query*|*fileBucket* +
_可选_|fileBucket|string
|*Query*|*fileId* +
_可选_|文件id|integer (int64)
|*Query*|*name* +
_可选_|文件的全名，使用URL编码（UTF-8），最多256个字符|string
|*Query*|*object* +
_必填_|object|string
|*Query*|*size* +
_可选_|size|integer (int64)
|*Query*|*sourceId* +
_可选_|调用方的文件源ID，不能重复|string
|*Body*|*callbackBody* +
_可选_|OSS回调file模块的回调体|<<_string,String>>
|===


===== 响应

[options="header", cols=".^2a,.^14a,.^4a"]
|===
|HTTP代码|说明|类型
|*200*|OK|<<_d9ca73ac43d7c6ced2c047acc5c261c5,GeneralResponse«FileBean»>>
|*404*|Not Found|无内容
|===

===== 生成

* `application/json`
* `\*/*`


===== HTTP请求示例

====== 请求 path
----
/oss/receive?object=string
----


====== 请求 header
[source,json]
----
"string"
----
