
[[_supportedfileformatsusingget]]
==== 支持的文件类型
....
GET /support
....


===== 说明
获取app对应支持的文件类型


===== 参数

[options="header", cols=".^2a,.^3a,.^9a,.^4a,.^2a"]
|===
|类型|名称|说明|类型|默认值
|*Header*|*Authorization* +
_必填_|authorization header|string|`"bearer dc671840-bacc-4dc5-a134-97c1918d664b"`
|===
===== Request Body
无
[source,json]
----
curl -X GET
-H "Authorization: bearer <your accessToken>"
-H "Content-Type: application/json"
"https://file.bimface.com/support"
----

===== 响应

[options="header", cols=".^2a,.^14a,.^4a"]
|===
|HTTP代码|说明|类型
|*200*|OK|<<_234fffd84e33fcdb03dd901be9f44d5c,GeneralResponse«SupportFileBean»>>
|*404*|Not Found|未找到链接
|===

===== HTTP请求示例

====== 请求 path
----
/support
----

====== 请求 header
[source,json]
----
"string"
----
