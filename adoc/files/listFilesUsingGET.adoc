
[[_listfilesusingget]]
==== 获取文件列表
....
GET /files
....

===== 说明
根据条件获取文件列表

===== 参数

[options="header", cols=".^2a,.^3a,.^9a,.^4a,.^2a"]
|===
|类型|名称|说明|类型|默认值
|*Header*|*Authorization* +
_必填_|authorization header|string|`"bearer dc671840-bacc-4dc5-a134-97c1918d664b"`
|*Query*|*endTime* +
_可选_|截止日期|string|
|*Query*|*offset* +
_可选_|查询结果偏移|integer (int64)|
|*Query*|*rows* +
_可选_|查询结果数,默认为100， 最大500|integer (int64)|
|*Query*|*scope* +
_可选_|查询文件范围|string|
|*Query*|*startTime* +
_可选_|起始日期|string|
|*Query*|*status* +
_可选_|文件状态|string|
|*Query*|*suffix* +
_可选_|文件后缀|string|
|===

===== HTTP请求示例

====== 请求 path
----
/files
----


====== 请求 header
[source,json]
----
"string"
----
