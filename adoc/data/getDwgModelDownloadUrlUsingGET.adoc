
[[_getdwgmodeldownloadurlusingget]]
==== 获取dwg模型zip包下载地址
....
GET /databag/dwg
....


===== 参数

[options="header", cols=".^2a,.^3a,.^9a,.^4a,.^2a"]
|===
|类型|名称|说明|类型|默认值
|*Header*|*Authorization* +
_必填_|authorization header|string|`"bearer "`
|*Query*|*fileId* +
_可选_|文件ID|integer (int64)|
|===
===== 生成

* `application/json`
* `\*/*`


===== HTTP请求示例

====== 请求 path
----
/databag/dwg
----


====== 请求 header
[source,json]
----
"string"
----


===== 响应

[options="header", cols=".^2a,.^14a,.^4a"]
|===
|HTTP代码|说明|类型
|*200*|OK|<<_geturlswaggerdisplay,GetUrlSwaggerDisplay>>
|*401*|Unauthorized|无内容
|*403*|Forbidden|无内容
|*404*|Not Found|无内容
|===
===== HTTP响应示例

====== 响应 200
[source,json]
----
{
  "code" : "success",
  "message" : null,
  "data" : "http://m.bimface.com/xxx.zip"
}
----
