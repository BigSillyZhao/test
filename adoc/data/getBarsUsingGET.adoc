
[[_getbarsusingget]]
==== 集成模型获取构件钢筋详情
....
GET /v2/integrations/{integrateId}/elements/{elementId}/bars
....


===== 参数

[options="header", cols=".^2a,.^3a,.^9a,.^4a,.^2a"]
|===
|类型|名称|说明|类型|默认值
|**Header**|**Authorization** +
__必填__|authorization header|string|`"bearer "`
|**Path**|**elementId** +
__必填__|elementId|string|
|**Path**|**integrateId** +
__必填__|integrateId|integer (int64)|
|===


===== 响应

[options="header", cols=".^2a,.^14a,.^4a"]
|===
|HTTP代码|说明|类型
|**200**|OK|<<_0ee283fce91cddb055d0ad764d2e1240,GeneralResponse«List«Bar»»>>
|**401**|Unauthorized|无内容
|**403**|Forbidden|无内容
|**404**|Not Found|无内容
|===

===== 生成

* `application/json`
* `\*/*`


===== HTTP请求示例

====== 请求 path
----
/v2/integrations/0/elements/string/bars
----


====== 请求 header
[source,json]
----
"string"
----
