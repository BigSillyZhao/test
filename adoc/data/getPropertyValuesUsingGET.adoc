
[[_getpropertyvaluesusingget]]
==== 构件搜索API支持返回指定属性的所有可能值
....
GET /v2/query/propertyValues
....


===== 参数

[options="header", cols=".^2a,.^3a,.^9a,.^4a,.^2a"]
|===
|类型|名称|说明|类型|默认值
|**Header**|**Authorization** +
__必填__|authorization header|string|`"bearer "`
|**Query**|**properties** +
__必填__|properties|< string > array(multi)|
|**Query**|**targetIds** +
__必填__|targetIds|< string > array(multi)|
|===


===== 响应

[options="header", cols=".^2a,.^14a,.^4a"]
|===
|HTTP代码|说明|类型
|**200**|OK|<<_a5a92298dd33dde410fee219f1e24ccf,GeneralResponse«List«PropertyValuesResp»»>>
|**401**|Unauthorized|无内容
|**403**|Forbidden|无内容
|**404**|Not Found|无内容
|===

===== 生成

* `application/json`
* `\*/*`


===== HTTP请求示例

====== 请求 path
----
/v2/query/propertyValues?properties=string&targetIds=string
----


====== 请求 header
[source,json]
----
"string"
----
