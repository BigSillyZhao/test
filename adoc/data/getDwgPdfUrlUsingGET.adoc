
[[_getdwgpdfurlusingget]]
==== 获取DWG文件转换生成的pdf文件的url
....
GET /v2/databag/pdf
....


===== 参数

[options="header", cols=".^2a,.^3a,.^9a,.^4a,.^2a"]
|===
|类型|名称|说明|类型|默认值
|*Header*|*Authorization* +
_必填_|authorization header|string|`"bearer "`
|*Query*|*dwgFileId* +
_可选_|DWG文件ID|integer (int64)|
|===


===== 响应

[options="header", cols=".^2a,.^14a,.^4a"]
|===
|HTTP代码|说明|类型
|*200*|OK|<<_7566474039fac907ea0573a8f68797fd,GeneralResponse«string»>>
|*401*|Unauthorized|无内容
|*403*|Forbidden|无内容
|*404*|Not Found|无内容
|===

===== 生成

* `application/json`
* `\*/*`


===== HTTP请求示例

====== 请求 path
----
/v2/databag/pdf
----


====== 请求 header
[source,json]
----
"string"
----


===== HTTP响应示例

====== 响应 200
[source,json]
----
{
    "code": "success",
    "message": null,
    "data": "https://m.bimface.com/c31a0438aa6f6dec4937ffecd9be4d74/resource/pdf/drawing.pdf"
}
----
