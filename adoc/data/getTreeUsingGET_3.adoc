
[[_gettreeusingget_3]]
==== 单模型构件分类树
....
GET /hierarchy
....


===== 参数

[options="header", cols=".^2a,.^3a,.^9a,.^4a,.^2a"]
|===
|类型|名称|说明|类型|默认值
|**Header**|**Authorization** +
__必填__|authorization header|string|`"bearer "`
|**Query**|**fileId** +
__必填__|fileId|integer (int64)|
|**Query**|**v** +
__可选__|v|string|
|===


===== 响应

[options="header", cols=".^2a,.^14a,.^4a"]
|===
|HTTP代码|说明|类型
|**200**|OK|<<_b80c1d2d7c7ac73beacfcf91f5090656,GeneralResponse«object»>>
|**401**|Unauthorized|无内容
|**403**|Forbidden|无内容
|**404**|Not Found|无内容
|===


===== 消耗

* `application/json`


===== 生成

* `application/json`
* `\*/*`


===== HTTP请求示例

====== 请求 path
----
/hierarchy?fileId=0
----


====== 请求 header
[source,json]
----
"string"
----

====== 响应 200
[source,json]
----
{

  "data" : "object",
  "message" : "message"
}
