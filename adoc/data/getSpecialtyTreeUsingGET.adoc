
[[_getspecialtytreeusingget]]
==== 集成模型默认专业分类树(v2.0)
....
GET /integration/tree/specialty
....


===== 参数

[options="header", cols=".^2a,.^3a,.^9a,.^4a,.^2a"]
|===
|类型|名称|说明|类型|默认值
|*Header*|*Authorization* +
_必填_|authorization header|string|`"bearer "`
|*Query*|*integrateId* +
_必填_|integrateId|integer (int64)|
|===

===== 生成

* `application/json`
* `\*/*`


===== HTTP请求示例

====== 请求 path
----
/integration/tree/specialty?integrateId=0
----


====== 请求 header
[source,json]
----
"string"
----
===== 响应

[options="header", cols=".^2a,.^14a,.^4a"]

|HTTP代码|说明|类型

|*401*|Unauthorized|无内容
|*403*|Forbidden|无内容
|*404*|Not Found|无内容
|===
