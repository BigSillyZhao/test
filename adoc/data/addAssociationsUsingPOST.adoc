
[[_addassociationsusingpost]]
==== 新增双向业务挂接,可以在request body中指定新增项，也可以指定从其他集成拷贝
....
POST /v2/integrations/{integrateId}/associations
....


===== 参数

[options="header", cols=".^2a,.^3a,.^9a,.^4a,.^2a"]
|===
|类型|名称|说明|类型|默认值
|*Header*|*Authorization* +
_必填_|authorization header|string|`"bearer "`
|*Path*|*integrateId* +
_必填_|集成ID|string|
|*Query*|*fromBusinessFlag* +
_可选_|指定拷贝源业务标志|string|
|*Query*|*fromBusinessType* +
_可选_|指定拷贝源业务类型|string|
|*Query*|*fromIntegrationId* +
_可选_|指定拷贝源集成ID|string|
|*Body*|*request* +
_可选_|指定新增双向业务挂接项|<<_businessassociationrequest,BusinessAssociationRequest>>|
|===


===== 响应

[options="header", cols=".^2a,.^14a,.^4a"]
|===
|HTTP代码|说明|类型
|*200*|OK|<<_generalresponse,GeneralResponse>>
|*201*|Created|无内容
|*401*|Unauthorized|无内容
|*403*|Forbidden|无内容
|*404*|Not Found|无内容
|===

===== 生成

* `application/json`
* `\*/*`


===== HTTP请求示例

====== 请求 path
----
/v2/integrations/string/associations
----


====== 请求 header
[source,json]
----
"string"
----


====== 请求 body
[source,json]
----
{
    "businessFlag": "test_flag1",
    "businessType": "test_type1",
    "associations": [
        {
            "businessId": "business_id_2",
            "elementId": "element_id_1"
        },
        {
            "businessId": "business_id_3",
            "elementId": "element_id_1"
        },
        {
            "businessId": "business_id_2",
            "elementId": "element_id_2"
        },
        {
            "businessId": "business_id_3",
            "elementId": "element_id_4"
        }
    ]
}
----
