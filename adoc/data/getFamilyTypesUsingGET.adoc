
[[_getfamilytypesusingget]]
==== rfa文件族类型列表
....
GET /family/familyType
....
====== 说明
注意：此接口只适用于rfa类型文件

===== 参数

[options="header", cols=".^2a,.^3a,.^9a,.^4a,.^2a"]
|===
|类型|名称|说明|类型|默认值
|*Header*|*Authorization* +
_必填_|authorization header|string|`"bearer "`
|*Query*|*fileId* +
_必填_|fileId|integer (int64)|
|===


===== 生成

* `application/json`
* `\*/*`


===== HTTP请求示例

====== 请求 path
----
/family/familyType?fileId=0
----


====== 请求 header
[source,json]
----
"string"
----

===== 响应

[options="header", cols=".^2a,.^14a,.^4a"]
|===
|HTTP代码|说明|类型
|*200*|OK|<<_f30a5c359780d80839e91369c364660c,GeneralResponse«List«RfaFamilyType»»>>
|*401*|Unauthorized|无内容
|*403*|Forbidden|无内容
|*404*|Not Found|无内容
|===
