
[[_getelementmaterialusingget]]
==== 单模型获取构件材质
....
GET /v2/files/{fileId}/elements/{elementId}/materials
....


===== 参数

[options="header", cols=".^2a,.^3a,.^9a,.^4a,.^2a"]
|===
|类型|名称|说明|类型|默认值
|*Header*|*Authorization* +
_必填_|authorization header|string|`"bearer "`
|*Path*|*elementId* +
_必填_|elementId|string|
|*Path*|*fileId* +
_必填_|fileId|integer (int64)|
|===


===== 生成

* `application/json`
* `\*/*`


===== HTTP请求示例

====== 请求 path
----
/v2/files/0/elements/string/materials
----


====== 请求 header
[source,json]
----
"string"
----
===== 响应

[options="header", cols=".^2a,.^14a,.^4a"]
|===
|HTTP代码|说明|类型
|*200*|OK|<<_0bb12676abfb558812ad4227ac76981a,GeneralResponse«List«MaterialInfo»»>>
|*401*|Unauthorized|无内容
|*403*|Forbidden|无内容
|*404*|Not Found|无内容
|===
