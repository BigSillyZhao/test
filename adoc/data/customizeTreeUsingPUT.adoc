
[[_customizetreeusingput]]
==== 自定义构造集成模型构件分类树
....
PUT /integration/tree
....


===== 参数

[options="header", cols=".^2a,.^3a,.^9a,.^4a,.^2a"]
|===
|类型|名称|说明|类型|默认值
|*Header*|*Authorization* +
_必填_|authorization header|string|`"bearer "`
|*Query*|*integrateId* +
_必填_|integrateId|integer (int64)|
|*Query*|*promoteHierarchy* +
_可选_|promoteHierarchy|boolean|
|*Body*|*customizeRequest* +
_必填_|customizeRequest|< <<_customizedtree,CustomizedTree>> > array|
|===


===== 生成

* `application/json`
* `\*/*`


===== HTTP请求示例

====== 请求 path
----
/integration/tree?integrateId=0
----


====== 请求 header
[source,json]
----
"string"
----


====== 请求 body
[source,json]
----
[
{
        "type": "specialty",
        "item": {
            "type": "category"
        }
    },
    {
        "type": "floor"
 }
]
----
