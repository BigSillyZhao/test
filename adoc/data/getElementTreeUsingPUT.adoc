
[[_getelementtreeusingput]]
==== 选定构件构造构件分类树
....
PUT /integration/tree/filter
....


===== 参数

[options="header", cols=".^2a,.^3a,.^9a,.^4a,.^2a"]
|===
|类型|名称|说明|类型|默认值
|*Header*|*Authorization* +
_必填_|authorization header|string|`"bearer "`
|*Query*|*integrateId* +
_必填_|integrateId|integer (int64)|
|*Body*|*request* +
_必填_|request|<<_elementstreerequest,ElementsTreeRequest>>|
|===

===== 生成

* `application/json`
* `\*/*`


===== HTTP请求示例

====== 请求 path
----
/integration/tree/filter?integrateId=0
----

----

----
