
[[_getintegratefilesusingget]]
==== 查询指定的集成模型内参与集成的子文件信息
....
GET /v2/integrations/{integrateId}/files
....


===== 参数

[options="header", cols=".^2a,.^3a,.^9a,.^4a,.^2a"]
|===
|类型|名称|说明|类型|默认值
|**Header**|**Authorization** +
__必填__|authorization header|string|`"bearer "`
|**Path**|**integrateId** +
__必填__|integrateId|integer (int64)|
|===


===== 响应

[options="header", cols=".^2a,.^14a,.^4a"]
|===
|HTTP代码|说明|类型
|**200**|OK|<<_56d84cb359fd9a04c5e5d12983f7a5af,GeneralResponse«List«IntegrateFile»»>>
|**401**|Unauthorized|无内容
|**403**|Forbidden|无内容
|**404**|Not Found|无内容
|===
===== 生成

* `application/json`
* `\*/*`


===== HTTP请求示例

====== 请求 path
----
/v2/integrations/0/files
----


====== 请求 header
[source,json]
----
"string"
----
