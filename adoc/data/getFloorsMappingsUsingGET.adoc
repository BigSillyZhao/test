
[[_getfloorsmappingsusingget]]
==== 单模型文件 id 与楼层信息映射
....
GET /v2/files/{fileIds}/fileIdfloorsMappings
....


===== 参数

[options="header", cols=".^2a,.^3a,.^9a,.^4a,.^2a"]
|===
|类型|名称|说明|类型|默认值
|*Header*|*Authorization* +
_必填_|authorization header|string|`"bearer "`
|*Query*|*fileIds* +
_必填_|fileIds|< string > array(multi)|
|===


===== 生成

* `application/json`
* `\*/*`


===== HTTP请求示例

====== 请求 path
----
/v2/files/{fileIds}/fileIdfloorsMappings?fileIds=string
----


====== 请求 header
[source,json]
----
"string"
----

===== 响应

[options="header", cols=".^2a,.^14a,.^4a"]
|===
|HTTP代码|说明|类型
|*200*|OK|<<_generalresponse,GeneralResponse>>
|*401*|Unauthorized|无内容
|*403*|Forbidden|无内容
|*404*|Not Found|无内容
|===
