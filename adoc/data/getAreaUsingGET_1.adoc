
[[_getareausingget_1]]
==== 集成模型模型获取Area属性
....
GET /v2/integrations/{integrateId}/areas/{areaId}
....


===== 参数

[options="header", cols=".^2a,.^3a,.^9a,.^4a,.^2a"]
|===
|类型|名称|说明|类型|默认值
|*Header*|*Authorization* +
_必填_|authorization header|string|`"bearer "`
|*Path*|*areaId* +
_必填_|areaId|string|
|*Path*|*integrateId* +
_必填_|integrateId|integer (int64)|
|===



===== 生成

* `application/json`
* `\*/*`


===== HTTP请求示例

====== 请求 path
----
/v2/integrations/0/areas/string
----


====== 请求 header
[source,json]
----
"string"
----

===== 响应

[options="header", cols=".^2a,.^14a,.^4a"]
|===
|HTTP代码|说明|类型
|*200*|OK|<<_e990d36191db025cb72e80467ff8b395,GeneralResponse«Area»>>
|*401*|Unauthorized|无内容
|*403*|Forbidden|无内容
|*404*|Not Found|无内容
|===
