
[[_gettreeusingget]]
==== 获取模型对比构件分类树
....
GET /v2/comparisons/{comparisonId}/tree
....


===== 参数

[options="header", cols=".^2a,.^3a,.^9a,.^4a,.^2a"]
|===
|类型|名称|说明|类型|默认值
|*Header*|*Authorization* +
_必填_|authorization header|string|`"bearer "`
|*Path*|*comparisonId* +
_必填_|comparisonId|integer (int64)|
|===


===== 响应

[options="header", cols=".^2a,.^14a,.^4a"]
|===
|HTTP代码|说明|类型
|*200*|OK|<<_108535eac2e175cccc8ed847ffaf3ec6,GeneralResponse«Tree»>>
|*401*|Unauthorized|无内容
|*403*|Forbidden|无内容
|*404*|Not Found|无内容
|===


===== 生成

* `application/json`
* `\*/*`


===== HTTP请求示例

====== 请求 path
----
/v2/comparisons/0/tree
----


====== 请求 header
[source,json]
----
"string"
----
