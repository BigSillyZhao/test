
[[_getassociationsbyelementidusingget]]
==== 根据构件ID查询对应业务挂接
....
GET /v2/integrations/{integrateId}/elements/{elementId}/associations
....


===== 参数

[options="header", cols=".^2a,.^3a,.^9a,.^4a,.^2a"]
|===
|类型|名称|说明|类型|默认值
|*Header*|*Authorization* +
_必填_|authorization header|string|`"bearer "`
|*Path*|*elementId* +
_必填_|构件ID|string|
|*Path*|*integrateId* +
_必填_|集成ID|string|
|*Query*|*businessFlag* +
_可选_|业务挂接标志|string|
|*Query*|*businessType* +
_可选_|业务挂接类型|string|
|===


===== 响应

[options="header", cols=".^2a,.^14a,.^4a"]
|===
|HTTP代码|说明|类型
|*200*|OK|<<_1f05a540b945e3dccfcada2613025abd,GeneralResponse«List«ElementBusinessAssociation»»>>
|*401*|Unauthorized|无内容
|*403*|Forbidden|无内容
|*404*|Not Found|无内容
|===


===== 生成

* `application/json`
* `\*/*`


===== HTTP请求示例

====== 请求 path
----
/v2/integrations/string/elements/string/associations
----


====== 请求 header
[source,json]
----
"string"
----
