
[[_getelementidsusingget]]
==== 单模型查询构件
....
GET /v2/files/{fileId}/elementIds
....


===== 参数

[options="header", cols=".^2a,.^3a,.^9a,.^4a,.^2a"]
|===
|类型|名称|说明|类型|默认值
|*Header*|*Authorization* +
_必填_|authorization header|string|`"bearer "`
|*Path*|*fileId* +
_可选_|文件ID|integer (int64)|
|*Query*|*areaId* +
_可选_||string|
|*Query*|*categoryId* +
_可选_||string|
|*Query*|*family* +
_可选_||string|
|*Query*|*familyType* +
_可选_||string|
|*Query*|*floor* +
_可选_||string|
|*Query*|*roomId* +
_可选_||string|
|*Query*|*specialty* +
_可选_||string|
|*Query*|*systemType* +
_可选_||string|
|===

===== 生成

* `application/json`
* `\*/*`


===== HTTP请求示例

====== 请求 path
----
/v2/files/0/elementIds
----


====== 请求 header
[source,json]
----
"string"
----
