
[[_pagegetmodelcompareresultusingget]]
==== 分页获取模型对比结果
....
GET /v2/comparisons/{comparisonId}/diff
....


===== 参数

[options="header", cols=".^2a,.^3a,.^9a,.^4a,.^2a"]
|===
|类型|名称|说明|类型|默认值
|**Header**|**Authorization** +
__必填__|authorization header|string|`"bearer "`
|**Path**|**comparisonId** +
__必填__|comparisonId|integer (int64)|
|**Query**|**elementName** +
__可选__|elementName|string|
|**Query**|**family** +
__可选__|family|string|
|**Query**|**page** +
__可选__|page|integer (int32)|
|**Query**|**pageSize** +
__可选__|pageSize|integer (int32)|
|===


===== 响应

[options="header", cols=".^2a,.^14a,.^4a"]
|===
|HTTP代码|说明|类型
|**200**|OK|<<_cb4300e5d2691ac6df8ebc7a8e6d5a4b,GeneralResponse«Pagination«ModelCompareDiff»»>>
|**401**|Unauthorized|无内容
|**403**|Forbidden|无内容
|**404**|Not Found|无内容
|===
===== 生成

* `application/json`
* `\*/*`


===== HTTP请求示例

====== 请求 path
----
/v2/comparisons/0/diff
----


====== 请求 header
[source,json]
----
"string"
----
