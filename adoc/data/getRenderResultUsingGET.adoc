
[[_getrenderresultusingget]]
==== 获取离线渲染结果图片
....
GET /render
....


===== 参数

[options="header", cols=".^2a,.^3a,.^9a,.^4a,.^2a"]
|===
|类型|名称|说明|类型|默认值
|**Header**|**Authorization** +
__必填__|authorization header|string|`"bearer "`
|**Query**|**fileId** +
__必填__|fileId|integer (int64)|
|===


===== 响应

[options="header", cols=".^2a,.^14a,.^4a"]
|===
|HTTP代码|说明|类型
|**200**|OK|< string (byte) > array
|**401**|Unauthorized|无内容
|**403**|Forbidden|无内容
|**404**|Not Found|无内容
|===


===== 生成

* `image/png`
* `application/json`


===== HTTP请求示例

====== 请求 path
----
/render?fileId=0
----


====== 请求 header
[source,json]
----
"string"
----
