
[[_getroomsusingget]]
==== 集成模型获取楼层对应房间列表
....
GET /v2/integrations/{integrateId}/rooms
....


===== 参数

[options="header", cols=".^2a,.^3a,.^9a,.^4a,.^2a"]
|===
|类型|名称|说明|类型|默认值
|**Header**|**Authorization** +
__必填__|authorization header|string|`"bearer "`
|**Path**|**integrateId** +
__必填__|integrateId|integer (int64)|
|**Query**|**floorId** +
__必填__|floorId|string|
|===


===== 响应

[options="header", cols=".^2a,.^14a,.^4a"]
|===
|HTTP代码|说明|类型
|**200**|OK|<<_36ffe49fd4b01b34a9cf20f19b018291,GeneralResponse«List«Room»»>>
|**401**|Unauthorized|无内容
|**403**|Forbidden|无内容
|**404**|Not Found|无内容
|===

===== 生成

* `application/json`
* `\*/*`


===== HTTP请求示例

====== 请求 path
----
/v2/integrations/0/rooms?floorId=string
----


====== 请求 header
[source,json]
----
"string"
----
