
[[_getfilethumbnailurlusingget]]
==== 获取缩略图url
....
GET /v2/databag/thumbnail
....


===== 参数

[options="header", cols=".^2a,.^3a,.^9a,.^4a,.^2a"]
|===
|类型|名称|说明|类型|默认值
|*Header*|*Authorization* +
_必填_|authorization header|string|`"bearer "`
|*Query*|*fileId* +
_必填_|文件ID|string|
|*Query*|*size* +
_可选_|缩略图大小|integer (int32)|
|===


===== 响应

[options="header", cols=".^2a,.^14a,.^4a"]
|===
|HTTP代码|说明|类型
|*200*|OK|<<_7566474039fac907ea0573a8f68797fd,GeneralResponse«string»>>
|*401*|Unauthorized|无内容
|*403*|Forbidden|无内容
|*404*|Not Found|无内容
|===

===== 生成

* `application/json`
* `\*/*`


===== HTTP请求示例

====== 请求 path
----
/v2/databag/thumbnail?fileId=string
----


====== 请求 header
[source,json]
----
"string"
----


===== HTTP响应示例

====== 响应 200
[source,json]
----
{
    "code": "success",
    "message": null,
    "data": "https://m.bimface.com/c31a0438aa6f6dec4937ffecd9be4d74/thumbnail/96.png"
}
----
