
[[_getfiledatabagdownloadurl2usingget]]
==== 获取单模型数据包下载地址
....
GET /databag/downloadUrl
....


===== 参数

[options="header", cols=".^2a,.^3a,.^9a,.^4a,.^2a"]
|===
|类型|名称|说明|类型|默认值
|**Header**|**Authorization** +
__必填__|authorization header|string|`"bearer "`
|**Query**|**databagVersion** +
__可选__|数据包版本；对于offline、vr数据包，如果只有一个，则下载唯一的数据包，如果多个，则必须指定数据包版本|string|
|**Query**|**fileId** +
__必填__|文件ID|string|
|**Query**|**type** +
__可选__|数据包类型，如offline、vr、igms|string|
|===


===== 响应

[options="header", cols=".^2a,.^14a,.^4a"]
|===
|HTTP代码|说明|类型
|**200**|OK|<<_geturlswaggerdisplay,GetUrlSwaggerDisplay>>
|**401**|Unauthorized|无内容
|**403**|Forbidden|无内容
|**404**|Not Found|无内容
|===

===== 生成

* `application/json`
* `\*/*`


===== HTTP请求示例

====== 请求 path
----
/databag/downloadUrl?fileId=string
----


====== 请求 header
[source,json]
----
"string"
----
