
[[_getroomusingget]]
==== 集成模型模型获取房间属性
....
GET /v2/integrations/{integrateId}/rooms/{roomId}
....


===== 参数

[options="header", cols=".^2a,.^3a,.^9a,.^4a,.^2a"]
|===
|类型|名称|说明|类型|默认值
|**Header**|**Authorization** +
__必填__|authorization header|string|`"bearer "`
|**Path**|**integrateId** +
__必填__|integrateId|integer (int64)|
|**Path**|**roomId** +
__必填__|roomId|string|
|===


===== 响应

[options="header", cols=".^2a,.^14a,.^4a"]
|===
|HTTP代码|说明|类型
|**200**|OK|<<_51d02355624740a98efc72eae7c359eb,GeneralResponse«Room»>>
|**401**|Unauthorized|无内容
|**403**|Forbidden|无内容
|**404**|Not Found|无内容
|===
===== 生成

* `application/json`
* `\*/*`


===== HTTP请求示例

====== 请求 path
----
/v2/integrations/0/rooms/string
----


====== 请求 header
[source,json]
----
"string"
----
